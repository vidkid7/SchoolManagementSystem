## Workflow Examples

### Marking Attendance
1. Navigate to Attendance section
2. Select class and subject
3. Select date
4. Mark students as Present/Absent/Late
5. Add remarks if needed
6. Submit attendance

### Entering Grades
1. Navigate to Examinations section
2. Select exam and subject
3. View student list
4. Enter marks for each student
5. Add comments if needed
6. Submit grades

### Creating Assignment
1. Navigate to Assignments section
2. Click "Create Assignment"
3. Fill in details:
   - Title
   - Description
   - Due date
   - Total marks
   - Attach files
4. Select classes
5. Publish assignment
