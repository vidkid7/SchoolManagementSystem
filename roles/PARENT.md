## Permissions Summary

### Parent Has:
- ✅ **View access** to child's academic data
- ✅ **Communication** with teachers
- ✅ **Fee payment** capabilities
- ✅ **Leave application** for child
- ✅ **Download** reports and certificates
- ✅ **Monitor** child's progress

### What Parent CANNOT Do:
- ❌ View other students' data
- ❌ Modify academic records
- ❌ Access administrative functions
- ❌ View staff information
- ❌ Modify system settings
- ❌ Submit assignments for child

## Workflow Examples

### Checking Child's Progress
1. Login to parent portal
2. View dashboard
3. Check attendance percentage
4. Review recent grades
5. Check pending assignments
6. View teacher remarks

### Paying School Fees
1. Navigate to Fee Management
2. View outstanding fees
3. Select invoice
4. Choose payment method
5. Complete payment
6. Download receipt

### Applying for Leave
1. Navigate to Attendance
2. Click "Apply for Leave"
3. Select dates
4. Enter reason
5. Upload supporting documents
6. Submit application
7. Track approval status

### Communicating with Teacher
1. Navigate to Messages
2. Select teacher
3. Compose message
4. Send message
5. Track response
