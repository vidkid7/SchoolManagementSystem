## Workflow Examples

### Creating a New ECA
1. Navigate to ECA Management
2. Click "Create ECA"
3. Enter ECA details:
   - Name
   - Category
   - Description
   - Coordinator
   - Schedule
   - Capacity
4. Save ECA
5. Publish to students

### Enrolling Students
1. Navigate to Enrollment
2. Select ECA
3. Search for students
4. Select students
5. Enroll students
6. Send confirmation

### Recording Achievement
1. Navigate to Achievements
2. Click "Record Achievement"
3. Select student
4. Select ECA
5. Enter achievement details
6. Upload certificate/photo
7. Save achievement
