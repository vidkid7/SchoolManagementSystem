## Getting Started

### System Requirements

- **Web Browser**: Chrome, Firefox, Safari, or Edge (latest version)
- **Internet Connection**: Stable internet for online features
- **Screen Resolution**: Minimum 1024x768 (responsive design supports mobile)

### Accessing the System

1. Open your web browser
2. Navigate to: `https://yourschool.edu.np`
3. You will see the login page

### First Time Login

1. Enter your username and password (provided by school admin)
2. Click "Sign In" button
3. You will be prompted to change your password on first login
4. Set a strong password (minimum 8 characters with letters and numbers)

## Admin Portal

### Dashboard

**Key Metrics:**

- Total Students
- Total Staff
- Pending Fees
- Attendance Rate
- Recent Activities

### Student Management

#### Adding a New Student

1. Navigate to **Students** ‚Üí **Add Student**
2. Fill in required information:
   - Personal Details (Name, DOB, Gender)
   - Contact Information
   - Guardian Details
   - Class & Section
   - Admission Date
3. Upload student photo (optional)
4. Click **Save**

#### Viewing Student List

1. Go to **Students** ‚Üí **Student List**
2. Use filters to search:
   - By Class
   - By Section
   - By Status (Active/Inactive)
3. Click on any student to view full profile

#### Editing Student Information

1. Find the student in Student List
2. Click **Edit** button
3. Update required fields
4. Click **Save Changes**

#### Student Promotion

1. Go to **Students** ‚Üí **Promotion**
2. Select current class and section
3. Select target class for promotion
4. Review student list
5. Click **Promote Selected Students**

### Staff Management

#### Adding Staff Members

1. Navigate to **Staff** ‚Üí **Add Staff**
2. Enter staff details:
   - Personal Information
   - Designation & Department
   - Qualification
   - Joining Date
   - Salary Information
3. Upload documents (certificates, ID proof)
4. Click **Save**

#### Managing Staff Documents

1. Go to staff profile
2. Click **Documents** tab
3. Upload required documents
4. Set expiry dates for time-sensitive documents
5. System will send reminders before expiry

### Class & Section Management

1. Navigate to **Academic** ‚Üí **Classes**
2. Click **Add Class**
3. Enter:
   - Class Name (e.g., "Class 10")
   - Grade Level
   - Section (A, B, C, etc.)
   - Class Teacher
   - Room Number
4. Click **Save**

### Fee Structure Setup

1. Go to **Finance** ‚Üí **Fee Structure**
2. Click **Add Fee Structure**
3. Select class
4. Add fee types:
   - Tuition Fee
   - Admission Fee
   - Exam Fee
   - Library Fee
   - Transport Fee
5. Set amounts and due dates
6. Click **Save**

### Generating Reports

1. Navigate to **Reports**
2. Select report type:
   - Student Report
   - Attendance Report
   - Fee Collection Report
   - Exam Results Report
3. Set date range and filters
4. Click **Generate Report**
5. Export as PDF or Excel

## FAQs

### General Questions

**Q: I forgot my password. What should I do?**
A: Click "Forgot Password?" on login page, enter your email, and follow the reset instructions sent to your email.

**Q: Can I access the system on my mobile phone?**
A: Yes! The system is fully responsive and works on all devices. You can also install it as a mobile app (PWA).

**Q: How do I change the language?**
A: Click the language switcher icon in the top-right corner to toggle between English and Nepali.

**Q: What browsers are supported?**
A: Chrome, Firefox, Safari, and Edge (latest versions). Chrome is recommended for best performance.

### For Students

**Q: How can I check my attendance?**
A: Go to Attendance section in your dashboard to view daily, monthly, and subject-wise attendance.

**Q: When will exam results be available?**
A: Results are published by teachers after grade entry. You'll receive a notification when results are available.

**Q: How do I pay my fees online?**
A: Go to Finance ‚Üí Pay Fees, select the invoice, choose payment method (eSewa/Khalti/IME Pay), and complete payment.

**Q: Can I download my report card?**
A: Yes! Go to Examinations ‚Üí Report Card, select the term, and click Download PDF.

### For Teachers

**Q: How do I mark attendance for multiple periods?**
A: Mark attendance separately for each period by selecting the period number before marking.

**Q: Can I edit attendance after submission?**
A: Yes, but only within 24 hours. After that, contact the admin for changes.

**Q: How do I enter marks for practical exams?**
A: Select the practical exam type when entering grades. The system supports theory and practical marks separately.

**Q: Can I send messages to all parents at once?**
A: Yes! In Communication ‚Üí Messages, select "All Parents of Class X" as recipients.

### For Parents

**Q: How do I track my child's attendance?**
A: Login to parent portal and go to Attendance section. You'll see daily attendance and monthly summary.

**Q: Will I be notified about fee due dates?**
A: Yes! You'll receive email and SMS reminders 7 days and 1 day before the due date.

**Q: Can I communicate with teachers?**
A: Yes! Use the Communication ‚Üí Messages feature to send messages to teachers.

**Q: How do I apply for leave for my child?**
A: Go to Leave ‚Üí Apply Leave, select your child, enter dates and reason, and submit.

### Technical Issues

**Q: The page is loading slowly. What should I do?**
A: Check your internet connection. Clear browser cache and cookies. Try using a different browser.

**Q: I'm getting a "Session Expired" error.**
A: Your login session has timed out. Please login again. Enable "Remember Me" for longer sessions.

**Q: Images are not loading properly.**
A: Check your internet connection. Try refreshing the page. Clear browser cache.

**Q: I can't upload documents.**
A: Ensure file size is under 10MB. Supported formats: PDF, JPG, PNG. Check your internet connection.
